# Modernization Implementation Plan

> **Historical note:** This project was originally a fork of `segmentio/socrates` and was renamed to **hiwrld** ([hiwrld.com](https://hiwrld.com)) on 2026-04-30 after the modernization landed. The plan body below uses the original "socrates" name throughout because that was the project's identity at the time of writing — preserved as the historical record of the work performed. All current code, branding, and documentation use the new name.

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bring the 2012-era segmentio/socrates markdown editor up to 2026 baselines: every dependency on a current, supported version; broken external CDN references replaced; Firebase v0 rewritten to the v11 modular SDK; ES modules + Vite build; Express 5 server.

**Architecture:** Stay structurally close to the original (Backbone views, single-page app, Firebase Realtime DB) — this is a modernization, not a rewrite. Introduce a Vite bundler so we can use the v9+ Firebase SDK (ESM-only) and replace the hand-concatenated `libs.js`. Replace abandoned Rainbow.js with highlight.js. Replace dead `cdn.mathjax.org` with `cdn.jsdelivr.net`. Upgrade Marked from its 2012 API (`marked()`, `sanitize:true`) to v15 with DOMPurify for sanitization. All other library bumps go through npm.

**Tech Stack:** Vite 5, Express 5, Backbone 1.6, Underscore 1.13, jQuery 3.7, Marked 15, DOMPurify 3, highlight.js 11, MathJax 4, Firebase 11, Dart Sass 1.83.

**Out of scope:** Replacing Backbone with React/Vue. Adding a test framework (this is a UI app with no existing tests; per user CLAUDE.md: verification is "run the actual flow and tell me what you saw"). Visual redesign. New features.

**Verification model:** No automated tests are added. Each phase ends with a manual browser smoke test against `npm run dev`. The final task is a full feature walkthrough with a screenshot.

```js
import _ from 'underscore';
import Backbone from 'backbone';
import { marked } from 'marked';
import DOMPurify from 'dompurify';
import hljs from 'highlight.js';

const $window = $(window);
const MINIMUM_WIDTH = 1000;

let hidden, visibilityChange;
if (typeof document.hidden !== 'undefined') {
  hidden = 'hidden';
  visibilityChange = 'visibilitychange';
} else if (typeof document.webkitHidden !== 'undefined') {
  hidden = 'webkitHidden';
  visibilityChange = 'webkitvisibilitychange';
}

export const View = Backbone.View.extend({

  youtubeEmbedTemplate: _.template(
    '<iframe width="100%" height="400" src="https://www.youtube.com/embed/<%= id %>" frameborder="0" allowfullscreen></iframe>'
  ),

  events: {
    'keyup .document-textarea': 'onTextareaKeyup',
    'click .read-only-button': 'onReadOnlyButtonClick',
    'click .write-only-button': 'onWriteOnlyButtonClick',
    'click .add-button': 'onAddButtonClick',
    'click .menu-button': 'onMenuButtonClick',
  },

  initialize(options) {
    _.bindAll(this, ...Object.keys(this.constructor.prototype).filter(k => typeof this[k] === 'function'));

    this.$title = this.$('title');
    this.$menu = this.$('.document-menu');
    this.$textarea = this.$('.document-textarea');
    this.$article = this.$('.document-article');
    this.$menuButton = this.$('.menu-button');
    this.$readOnlyButton = this.$('.read-only-button');
    this.$writeOnlyButton = this.$('.write-only-button');

    // jquery-tabby was a tiny custom plugin; replace with a one-liner if needed,
    // or leave it removed (modern browsers handle tab in textarea via keydown).
    // For now, install a minimal handler that inserts 4 spaces on Tab.
    this.$textarea.on('keydown', (event) => {
      if (event.key !== 'Tab') return;
      event.preventDefault();
      const el = event.target;
      const start = el.selectionStart;
      const end = el.selectionEnd;
      const tab = '    ';
      el.value = el.value.substring(0, start) + tab + el.value.substring(end);
      el.selectionStart = el.selectionEnd = start + tab.length;
    });

    // Defer document menu wiring to where it was — import lazily to avoid cycle.
    import('../document/document-menu-view.js').then(({ DocumentMenuView }) => {
      this.documentMenu = new DocumentMenuView({
        collection: this.model.get('documents'),
        el: this.$menu,
      }).on('select', this.onDocumentMenuSelect);
      this.documentMenu.render();
    });

    this.model
      .on('change:document', this.onAppDocumentChange)
      .on('change:state', this.renderState);
    if (this.model.has('document')) this.applyDocumentEventHandlers(this.model.get('document'));

    this.save = _.debounce(this.save, 500);
    this.renderYoutubeFilter = _.debounce(this.renderYoutubeFilter, 1000);

    $window.on('resize', this.onWindowResize);

    if (visibilityChange) {
      document.addEventListener(visibilityChange, () => {
        document[hidden] ? this.stopTitleCursor() : this.startTitleCursor();
      }, false);
    }
    this.startTitleCursor();
  },

  applyDocumentEventHandlers(doc, unbind) {
    const method = unbind ? 'off' : 'on';
    doc[method]('change:body', this.onDocumentBodyChange);
  },

  render() {
    this.renderTextarea().renderArticle().renderState();
    return this;
  },

  renderTextarea() {
    if (!this.model.has('document')) return this;
    this.$textarea.val(this.model.get('document').get('body'));
    return this;
  },

  renderArticle() {
    if (!this.model.has('document')) return this;
    const body = this.model.get('document').get('body');
    // marked v15: marked.parse(); sanitize via DOMPurify (replaces removed sanitize:true).
    const rawHtml = marked.parse(body);
    const cleanHtml = DOMPurify.sanitize(rawHtml);
    const quoted = this.renderSmartQuoteFilter(cleanHtml);
    this.$article.html(quoted);

    this.renderYoutubeFilter();
    this.renderCodeHighlightingFilter();
    this.renderMathJax();
    return this;
  },

  renderState() {
    const state = this.model.get('state');
    const readonly = state === 'read';
    const writeonly = state === 'write';

    if ($window.width() < MINIMUM_WIDTH && !state) return this.model.set('state', 'write');

    this.$readOnlyButton.state('pressed', readonly);
    this.$writeOnlyButton.state('pressed', writeonly);
    this.$el.state('write-only', writeonly);
    this.$el.state('read-only', readonly);
  },

  renderSmartQuoteFilter(markdown) {
    const leftSingleQuote = /([^\w])&#39;/g;
    const leftDoubleQuote = /([^\w])&quot;/g;
    const singleQuote = /&#39;/g;
    const doubleQuote = /&quot;/g;
    return markdown
      .replace(leftSingleQuote, '$1‘')
      .replace(leftDoubleQuote, '$1“')
      .replace(singleQuote, '’')
      .replace(doubleQuote, '”');
  },

  renderYoutubeFilter() {
    const self = this;
    this.$article.find('a[href*="youtube.com/watch?v="]').each(function () {
      const youtubeId = this.href.match(/\?v=([\w-]+)/)[1];
      const embed = self.youtubeEmbedTemplate({ id: youtubeId });
      $(this).replaceWith(embed);
    });
  },

  // MathJax v4: typesetPromise replaces Hub.Queue
  renderMathJax: _.debounce(function () {
    if (window.MathJax && window.MathJax.typesetPromise) {
      window.MathJax.typesetPromise([this.$article[0]]).catch(() => {});
    }
  }, 150),

  // highlight.js replaces Rainbow.js. Marked v15 emits class="language-xxx" already.
  renderCodeHighlightingFilter() {
    this.$article.find('pre code').each(function () {
      try { hljs.highlightElement(this); } catch (e) {}
    });
  },

  save() {
    this.model.get('document').save();
  },

  toggleMenu() {
    this.$menu.slideToggle();
    this.$menuButton.toggleState('pressed');
  },

  startTitleCursor() {
    this._cursorInterval = setInterval(() => this.renderTitleCursor(), 500);
  },

  stopTitleCursor() {
    clearInterval(this._cursorInterval);
    this.$title.html('Socrates');
  },

  renderTitleCursor() {
    this._titleCursor || (this._titleCursor = 'on');
    const cursor = this._titleCursor === 'on' ? '|' : '';
    this.$title.html('Socrates' + cursor);
    this._titleCursor = this._titleCursor === 'on' ? 'off' : 'on';
    return this;
  },

  onWindowResize() {
    if (this.model.has('state')) return;
    if ($window.width() < MINIMUM_WIDTH) this.model.set('state', 'write');
  },

  onTextareaKeyup() {
    this.model.get('document').set('body', this.$textarea.val());
    this.save();
  },

  onMenuButtonClick(event) {
    event.preventDefault();
    this.toggleMenu();
  },

  onAddButtonClick() {
    this.model.set('document', this.model.createDocument());
  },

  onReadOnlyButtonClick(event) {
    event.preventDefault();
    const state = this.$readOnlyButton.state('pressed') ? null : 'read';
    this.model.set('state', state);
  },

  onWriteOnlyButtonClick(event) {
    event.preventDefault();
    const state = this.$writeOnlyButton.state('pressed') ? null : 'write';
    this.model.set('state', state);
  },

  onAppDocumentChange(model, doc) {
    const previousDocument = this.model.previous('document');
    if (previousDocument) this.applyDocumentEventHandlers(previousDocument, true);
    this.applyDocumentEventHandlers(doc);
    this.renderTextarea().renderArticle();
  },

  onDocumentBodyChange() {
    this.renderTextarea().renderArticle();
  },

  onDocumentMenuSelect(menu, doc) {
    this.model.set('document', doc);
  },
});
```

```js
import $ from 'jquery';
import _ from 'underscore';
import Backbone from 'backbone';
import { DocumentCollection } from '../document/document-collection.js';
import { DocumentModel } from '../document/document-model.js';

const ID_CHARACTERS = 'abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ1234567890';
const ID_LENGTH = 7;

export const Model = Backbone.Model.extend({
  defaults: { document: null, documents: null, state: null },
  bookmarkKey: 'socrates.bookmarks',

  initialize() {
    _.bindAll(this, ...Object.keys(this.constructor.prototype).filter(k => typeof this[k] === 'function'));
    this.initializeRouter();
    this.initializeDocuments();
    this.on('change:document', this.onDocumentOrStateChange)
        .on('change:state', this.onDocumentOrStateChange);
  },

  initializeRouter() {
    this.router = new Backbone.Router({
      routes: { '': 'home', ':id(/:state)': 'document' },
    })
      .on('route:home', this.onHomeRoute)
      .on('route:document', this.onDocumentRoute);
  },

  initializeDocuments() {
    const documents = new DocumentCollection()
      .on('add remove', this.writeBookmarks)
      .on('remove', this.onDocumentRemove);
    this.set('documents', documents, { silent: true });
    _.each(this.readBookmarks(), this.fetchDocument);
  },

  fetchDocument(id, body) {
    if (!_.isString(body)) body = '';
    const doc = new DocumentModel({ id, body });
    this.get('documents').add(doc);
    return doc;
  },

  createDocument(body) {
    return this.fetchDocument(this.generateDocumentId(), body);
  },

  readBookmarks() {
    const bookmarkString = localStorage.getItem(this.bookmarkKey);
    return bookmarkString ? bookmarkString.split(',') : [];
  },

  writeBookmarks() {
    const ids = this.get('documents').map(d => d.id);
    localStorage.setItem(this.bookmarkKey, ids.join(','));
  },

  onHomeRoute() {
    let body = '';
    if (!localStorage.getItem(this.bookmarkKey)) body = onboarding;
    this.set('document', this.createDocument(body));
  },

  onDocumentRoute(id, state) {
    let doc = this.get('documents').find(d => id === d.id);
    if (!doc) doc = this.fetchDocument(id);
    this.set('document', doc);
    if (state === 'read' || state === 'write') this.set('state', state);
  },

  onDocumentOrStateChange() {
    if (!this.has('document')) return;
    const doc = this.get('document');
    const state = this.get('state');
    let fragment = doc.id;
    if (state) fragment += '/' + state;
    this.router.navigate(fragment, { replace: true });
  },

  onDocumentRemove(removedDocument) {
    const openDocument = this.get('document');
    if (openDocument.id !== removedDocument.id) return;
    const doc = this.get('documents').last() || this.createDocument();
    this.set('document', doc);
  },

  generateDocumentId() {
    let id = '';
    for (let x = 0; x < ID_LENGTH; x++) {
      id += ID_CHARACTERS.charAt(Math.floor(Math.random() * 62));
    }
    return id;
  },
});
```

```js
    _.bindAll(this, 'render', 'renderItem', 'onClickLi', 'onDeleteButtonClick');
    this.itemTemplate = _.template($('#document-menu-item-template').html() || '');
    this.collection.on('add remove reset change', this.render);
  },

  render() {
    this.$el.empty();
    let items = '';
    this.collection.each((doc) => {
      items += this.renderItem(doc);
    });
    this.$el.append(items);
    return this;
  },

  renderItem(doc) {
    return this.itemTemplate({ document: doc });
  },

  onClickLi(event) {
    event.preventDefault();
    const id = $(event.currentTarget).attr('data-id');
    const doc = this.collection.find(d => id === d.id);
    if (doc) this.trigger('select', this, doc);
  },

  onDeleteButtonClick(event) {
    event.preventDefault();
    const id = $(event.currentTarget).closest('li').attr('data-id');
    const doc = this.collection.find(d => id === d.id);
    if (doc) this.collection.remove(doc);
  },
});
```

- [ ] **Step 5c: Port `jquery-state` plugin to `src/lib/jquery-state.js`**

The original `libs/jquery-state.min.js` was a 1.6KB plugin that adds `$.fn.state(name, bool)`, `$.fn.toggleState(name)`, `$.fn.removeState(name)` — used in app-view.js for ARIA-aware class toggles (`pressed`, `write-only`, `read-only`). Rather than rewrite all 6 call sites, port a slim version:

Create `src/lib/jquery-state.js`:
```js
import $ from 'jquery';

const ariaMap = {
  'pressed': 'aria-pressed',
  'checked': 'aria-checked',
  'disabled': 'aria-disabled',
  'expanded': 'aria-expanded',
  'hidden': 'aria-hidden',
  'selected': 'aria-selected',
};

$.fn.state = function (name, value) {
  // Getter
  if (value === undefined) return this.hasClass(name);
  // Setter
  const truthy = !!value;
  if (truthy) this.addClass(name);
  else this.removeClass(name);
  if (ariaMap[name]) this.attr(ariaMap[name], truthy);
  return this;
};

$.fn.toggleState = function (name) {
  return this.state(name, !this.hasClass(name));
};

$.fn.removeState = function (name) {
  this.removeClass(name);
  if (ariaMap[name]) this.removeAttr(ariaMap[name]);
  return this;
};
```

Then import once in `src/main.js` (add to top, after the jQuery import):
```js
import './lib/jquery-state.js';
```

Note: this is a deliberately reduced port — it only handles the `pressed`/`write-only`/`read-only` cases the app actually uses, not the full original plugin's loading/required/invalid state machine. If a regression appears in ARIA behavior, expand `ariaMap` to match the original's full state list.

```js
import $ from 'jquery';
import _ from 'underscore';
import Backbone from 'backbone';
import { marked } from 'marked';
import { ref, onValue, set as fbSet } from 'firebase/database';
import { db } from '../firebase.js';

const days = ['Sunday','Monday','Tuesday','Wednesday','Thursday','Friday','Saturday'];
const months = ['January','February','March','April','May','June','July','August','September','October','November','December'];
const nths = { 1:'st', 2:'nd', 3:'rd', 21:'st', 22:'nd', 23:'rd', 31:'st' };

export const DocumentModel = Backbone.Model.extend({
  defaults() {
    return { created: new Date(), title: 'Untitled', body: '' };
  },

  initialize() {
    _.bindAll(this, 'generateTitle', 'save');
    this.generateTitle();
    this.initializeFirebase();
    this.on('change:body', this.generateTitle);
  },

  generateTitle() {
    const created = this.has('created') ? new Date(this.get('created')) : new Date();
    const day = days[created.getDay()];
    const month = months[created.getMonth()];
    const date = created.getDate();
    const nth = nths[date] || 'th';
    const year = created.getFullYear();
    let title = `Untitled - ${day}, ${month} ${date}${nth}, ${year}`;

    const body = this.get('body');
    const headings = $(marked.parse(body)).filter('h1, h2, h3, h4, h5, h6');
    if (headings.length > 0) title = $(headings[0]).text();

    this.set('title', title);
  },

  initializeFirebase() {
    if (!db) return; // no config — operate in local-only mode
    this._ref = ref(db, 'documents/' + this.id);
    onValue(this._ref, (snapshot) => {
      const val = snapshot.val();
      if (val) this.set(val);
    });
  },

  save() {
    if (!this._ref) return;
    fbSet(this._ref, this.toJSON());
  },
});
```

## Risks

1. **Firebase project with old data** — the original `socrates.firebaseio.com` is gone. There's no way to recover original docs. The app starts fresh.
2. **Backbone view options handling** — Backbone 1.0+ stopped automatically attaching `options` to `this`. This codebase doesn't use `this.options` directly (each `initialize` reads attributes via `this.model`/`this.collection`), so it shouldn't matter. If a regression appears, the fix is `this.options = options` at the top of `initialize`.
3. **MathJax v4 timing** — v4 is async-first. We disabled auto-typeset (`startup.typeset: false`) and call `typesetPromise` explicitly after each render. If the script hasn't finished loading yet on first render, the typeset is silently skipped. Acceptable: math will appear on the next text edit. If it's a problem, await `MathJax.startup.promise` once at app boot before first render.
4. **Smart-quote filter operates on HTML now, not the marked output** — The original ran the smart-quote filter on the raw HTML from `marked()` (which had `&#39;` entities). Marked v15 still produces `&#39;`/`&quot;` for quotes inside text by default, so the existing regexes should keep working. Verify in browser.
